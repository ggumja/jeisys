# 발송 템플릿 정리 (Jeisys 알림톡/SMS)

카카오 알림톡 템플릿 심사 등록 및 시스템 연동을 위해, 현재 시스템(Jeisys)에서 사용 중인 발송 상황별 템플릿(문구)을 정리한 문서입니다.

> **💡 개발자 참고 사항 (치환 변수)**
> 본문에 사용된 `{{변수명}}` 형태의 문구는 발송 시 백엔드 시스템(`templateParser.js`)을 통해 실제 데이터로 동적 치환됩니다.
> - `{{shop_name}}`: 쇼핑몰 이름 (예: (주)제이시스메디칼)
> - `{{customer_name}}`: 고객 이름
> - `{{order_number}}`: 주문 번호
> - `{{payment_amount}}`: 결제/입금 금액
> - 기타: `{{bank_name}}`, `{{vact_bank}}`, `{{vact_account}}`, `{{shipped_items}}` 등

## 1. 관리자 알림 (Admin Notifications)

### 1.1 새 주문 접수 (`sms_admin_new_order`)
- **발송 상황**: 고객이 쇼핑몰에서 새로운 주문을 완료했을 때 담당자에게 발송
- **문구 내용**:
  ```text
  [새 주문 접수]
  {{customer_name}}님으로부터 새로운 주문이 접수되었습니다.
  - 주문번호: {{order_number}}
  - 금액: {{payment_amount}}원
  ```

### 1.2 주문 취소 (`sms_admin_cancel_order`)
- **발송 상황**: 고객이 주문을 취소했을 때 담당자에게 발송
- **문구 내용**:
  ```text
  [주문 취소]
  {{customer_name}}님의 주문({{order_number}})이 취소되었습니다.
  ```

### 1.3 주문/결제 실패 (`sms_admin_failed_order`)
- **발송 상황**: 주문 또는 결제 과정에서 오류가 발생하여 실패했을 때
- **문구 내용**:
  ```text
  [주문/결제 실패]
  {{customer_name}}님의 결제가 실패하였습니다.
  - 주문번호: {{order_number}}
  ```

### 1.4 교환 요청 (`sms_admin_exchange_request`)
- **발송 상황**: 고객이 상품 교환을 요청했을 때
- **문구 내용**:
  ```text
  [교환 요청]
  {{customer_name}}님으로부터 교환 요청이 접수되었습니다.
  - 주문번호: {{order_number}}
  ```

### 1.5 반품 요청 (`sms_admin_return_request`)
- **발송 상황**: 고객이 상품 반품을 요청했을 때
- **문구 내용**:
  ```text
  [반품 요청]
  {{customer_name}}님으로부터 반품 요청이 접수되었습니다.
  - 주문번호: {{order_number}}
  ```

## 2. 고객 알림 - 결제 및 입금 대기 (Customer - Payment)

### 2.1 가상계좌 입금 대기 (`sms_cust_vact_waiting`)
- **발송 상황**: 고객이 가상계좌 결제를 선택하여 주문을 접수한 후, 입금을 기다릴 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  저희 쇼핑몰을 찾아주셔서 진심으로 감사드립니다! 💖

  주문하신 내역이 정상적으로 접수되었습니다.
  아래 계좌로 입금해 주시면 결제가 완전히 완료됩니다.

  ▶ 주문번호 : {{order_number}}
  ▶ 결제금액 : {{payment_amount}}원
  ▶ 입금은행 : {{vact_bank}}
  ▶ 계좌번호 : {{vact_account}}

  기한 내 입금되지 않을 경우 주문이 취소될 수 있으니 꼭 확인 부탁드릴게요. 오늘도 행복한 하루 보내세요! 😊
  ```

### 2.2 무통장 입금 대기 (`sms_cust_bank_waiting`)
- **발송 상황**: 고객이 무통장입금을 선택하여 주문을 접수한 후, 입금을 기다릴 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  저희 쇼핑몰을 찾아주셔서 진심으로 감사드립니다! 💖

  주문하신 내역이 정상적으로 접수되었습니다.
  아래 계좌로 입금해 주시면 결제가 완전히 완료됩니다.

  ▶ 주문번호 : {{order_number}}
  ▶ 결제금액 : {{payment_amount}}원
  ▶ 입금은행 : {{bank_name}}
  ▶ 계좌번호 : {{bank_account}}
  ▶ 예금주명 : {{bank_depositor}}

  입금자명이 다를 경우 결제 확인이 지연될 수 있으니 유의해 주세요. 오늘도 행복한 하루 보내세요! 😊
  ```

## 3. 고객 알림 - 결제 완료 (Customer - Payment Completed)

### 3.1 카드 결제 완료 (`sms_cust_order_complete_card`)
- **발송 상황**: 신용카드 등 즉시 결제 수단으로 결제가 성공적으로 완료되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  고객님의 소중한 주문이 정상적으로 결제 완료되었습니다! 🛍️

  저희 쇼핑몰을 이용해 주셔서 진심으로 감사드리며, 
  상품이 안전하게 도착할 수 있도록 꼼꼼히 준비해서 보내드리겠습니다.

  ▶ 주문번호 : {{order_number}}
  ▶ 결제금액 : {{payment_amount}}원

  배송이 시작되면 송장 번호와 함께 다시 안내해 드릴게요. 감사합니다! 🚚✨
  ```

### 3.2 가상계좌 입금 확인 (`sms_cust_order_complete_vact`)
- **발송 상황**: 발급된 가상계좌로 고객이 입금을 완료하여 결제가 확인되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  보내주신 결제 대금이 정상적으로 입금 확인되었습니다! 💸

  주문하신 상품은 정성껏 준비하여 빠르게 배송해 드릴 예정입니다.

  ▶ 주문번호 : {{order_number}}
  ▶ 입금금액 : {{payment_amount}}원

  배송이 시작되면 송장 번호와 함께 다시 안내해 드릴게요. 감사합니다! 🚚✨
  ```

### 3.3 무통장 입금 확인 (`sms_cust_order_complete_bank`)
- **발송 상황**: 무통장 입금 계좌로 고객이 입금을 완료하여 관리자가 이를 확인(승인)했을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  보내주신 결제 대금이 정상적으로 무통장 입금 확인되었습니다! 💸

  주문하신 상품은 정성껏 준비하여 빠르게 배송해 드릴 예정입니다.

  ▶ 주문번호 : {{order_number}}
  ▶ 입금금액 : {{payment_amount}}원
  ▶ 입금은행 : {{bank_name}}

  배송이 시작되면 송장 번호와 함께 다시 안내해 드릴게요. 감사합니다! 🚚✨
  ```

### 3.4 일부 금액 결제 완료 (`sms_cust_payment_partial`)
- **발송 상황**: 복합 결제나 분할 결제 시 일부 금액만 우선 결제(입금) 처리되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  주문에 대한 일부 금액 결제가 성공적으로 처리되었습니다. 💸

  ▶ 결제된 금액 : {{partial_paid_amount}}원
  ▶ 남은 잔금 : {{remaining_amount}}원

  남은 잔금을 모두 결제해 주셔야 상품 준비가 시작됩니다. 감사합니다.
  ```

### 3.5 잔금 포함 최종 결제 완료 (`sms_cust_payment_all_done`)
- **발송 상황**: 일부 결제 상태에서 남은 잔금까지 모두 지불되어 최종 결제가 완료되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  잔금 결제까지 모두 성공적으로 완료되었습니다! 🎉

  이제 주문하신 상품의 준비를 시작하며, 배송이 시작되면 다시 한번 안내해 드리겠습니다. 감사합니다.
  ```

## 4. 고객 알림 - 배송 (Customer - Shipping)

### 4.1 일반 상품 발송 (`sms_cust_shipping_standard`)
- **발송 상황**: 주문한 상품이 택배사로 인계되어 배송이 시작되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  주문하신 상품이 발송되었습니다. 기다려 주셔서 감사합니다! 📦

  ▶ 발송된 상품 : {{shipped_items}}
  ▶ 배송 정보 : {{courier_name}} ({{tracking_number}})

  상품이 안전하게 도착할 수 있도록 최선을 다하겠습니다.
  ```

### 4.2 분할 배송 (일부 상품 선발송) (`sms_cust_shipping_split`)
- **발송 상황**: 주문 상품 중 재고가 확보된 일부 상품만 먼저 발송할 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  주문하신 상품 중 준비된 일부 상품이 먼저 발송되었습니다. 📦

  ▶ 먼저 출발한 상품 : {{shipped_items}}
  ▶ 조금 더 기다리실 상품 : {{remaining_items}}

  나머지 상품도 입고되는 즉시 가장 빠르게 보내드리겠습니다!
  ```

### 4.3 잔여 상품 발송 (`sms_cust_shipping_remain`)
- **발송 상황**: 분할 배송 후 미발송되었던 나머지 상품의 배송이 시작되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  오래 기다려 주셔서 감사합니다! 미발송되었던 나머지 상품이 오늘 모두 출발했습니다. 📦

  ▶ 오늘 발송된 상품 : {{shipped_items}}

  이제 모든 상품이 발송되었습니다. 저희 쇼핑몰을 믿고 기다려주셔서 진심으로 감사드립니다.
  ```

## 5. 고객 알림 - 취소/교환/반품 (Customer - Claim)

### 5.1 교환 접수 (`sms_cust_exchange_received`)
- **발송 상황**: 고객의 상품 교환 요청이 시스템에 정상적으로 접수되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  요청하신 상품 교환 접수가 완료되었습니다. 🔄

  상품 이용에 불편을 드려 대단히 죄송하며, 회수 및 검수를 거쳐 최대한 신속하게 새 상품으로 교환해 드리겠습니다.
  ▶ 주문번호 : {{order_number}}

  포장된 상품을 기사님께 잘 전달해 주시면 감사하겠습니다.
  ```

### 5.2 반품 접수 (`sms_cust_return_received`)
- **발송 상황**: 고객의 반품(환불) 요청이 시스템에 정상적으로 접수되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  요청하신 상품 반품 접수가 완료되었습니다. ↩️

  상품 이용에 불편을 드려 대단히 죄송하며, 회수 및 검수를 거쳐 빠르게 환불 처리해 드리겠습니다.
  ▶ 주문번호 : {{order_number}}

  포장된 상품을 기사님께 잘 전달해 주시면 감사하겠습니다.
  ```

### 5.3 교환 처리 완료 (`sms_cust_exchange_done`)
- **발송 상황**: 회수된 상품 검수 후 새 상품으로 발송 처리가 완료되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  오래 기다리셨습니다! 요청하신 상품 교환 처리가 무사히 완료되어 새 상품으로 발송해 드렸습니다. ✅

  이번에는 꼭 만족스러운 상품이 되시길 바라며, 저희 쇼핑몰을 이용해 주셔서 감사합니다!
  ```

### 5.4 반품/환불 완료 (`sms_cust_return_done`)
- **발송 상황**: 반품 상품 회수 및 검수 후 최종 환불 처리가 승인되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  요청하신 반품 및 환불 처리가 정상적으로 완료되었습니다. 💳

  결제 수단에 따라 환불 완료까지 영업일 기준 3~7일이 소요될 수 있는 점 양해 부탁드립니다. 다음에는 더 좋은 상품으로 찾아뵙겠습니다. 감사합니다!
  ```

## 6. 고객 알림 - 회원 가입 및 계정 (Customer - Membership)

### 6.1 회원가입 환영 (`sms_cust_signup`)
- **발송 상황**: 신규 회원이 가입을 완료했을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  저희 쇼핑몰의 회원이 되신 것을 진심으로 환영합니다! 🎉

  고객님을 위한 다양하고 특별한 혜택들이 준비되어 있으니, 지금 바로 놀러오셔서 즐거운 쇼핑을 경험해 보세요! 🎁
  ```

### 6.2 임시 비밀번호 발급 (`sms_cust_password_reset`)
- **발송 상황**: 비밀번호 찾기 등을 통해 임시 비밀번호가 발급되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  고객님의 요청에 의해 임시 비밀번호가 성공적으로 발급되었습니다. 🔒

  보안을 위해 사이트에 로그인하신 후, 반드시 [마이페이지 > 회원정보 수정]에서 안전한 새 비밀번호로 변경해 주세요!
  ```

### 6.3 회원가입 승인 (`sms_cust_signup_approve`)
- **발송 상황**: 회원가입 신청이 최종 승인(APPROVED)되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  회원가입 신청이 최종 승인되었습니다! 🎉

  지금 바로 로그인하셔서 다양한 회원 혜택과 프리미엄 서비스를 이용해 보세요.
  ```

### 6.4 회원가입 반려 (`sms_cust_signup_reject`)
- **발송 상황**: 회원가입 신청이 반려(REJECTED)되었을 때
- **문구 내용**:
  ```text
  [ {{shop_name}} ]
  안녕하세요 {{customer_name}}님, 
  회원가입 신청이 반려되었음을 안내해 드립니다. ✉️

  ▶ 반려 사유 : {{reject_reason}}
  ```
