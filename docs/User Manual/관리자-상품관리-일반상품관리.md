# 📁 관리자 - 상품관리 - 일반상품관리 매뉴얼

이 문서는 제이시스 쇼핑몰의 **단일 상품(일반 소모품)**을 등록하고 관리하는 방법을 설명합니다. 기본 정보 입력부터 정기배송 할인, 재고 공유 설정까지 모든 기능을 상세히 다룹니다.

---

## 🚀 1. 상품 등록 시작
**메뉴 경로**: `관리자 > 상품 관리 > 상품 등록` (또는 기존 상품의 '수정' 버튼 클릭)

---

## 🛒 2. 이미지 및 기본 정보 설정

### 2.1 상품 이미지
*   **대표 이미지**: 상품 리스트와 상세 페이지 상단에 노출될 메인 이미지입니다. (권장: 800x800px)
*   **추가 이미지**: 상세 페이지 하단 슬라이드에 노출될 추가 사진들입니다. 최대 10개까지 업로드 가능합니다.

### 2.2 기본 정보 입력
*   **상품명 (필수)**: 고객에게 노출될 정식 명칭을 입력합니다.
*   **카테고리 (필수)**: 상품이 속할 카테고리를 선택합니다. 계층형으로 구성되어 있어 하위 카테고리까지 정확히 지정해야 합니다.
*   **상품코드(SKU) / SAP SKU**: 내부 물류 및 정산용 코드를 입력합니다. SKU는 중복될 수 없습니다.
*   **판매가 (필수)**: 숫자로 입력하면 자동으로 세 자릿수 콤마(,)가 적용됩니다.

---

## 📦 3. 재고 및 판매 정책

### 3.1 실시간 재고 관리
*   **재고 수량**: 현재 판매 가능한 수량을 입력합니다.
*   **판매 상태**: '판매중' 또는 '판매중지'를 설정합니다.
*   **노출 여부**: 체크 해제 시 관리자에게만 보이고 사용자 쇼핑몰 화면에서는 숨겨집니다.

### 3.2 재고 공유 설정 (마스터 상품 연동)
동일한 제품이나 용량만 다른 경우, 하나의 마스터 상품 재고를 공유할 수 있습니다.
*   **마스터 상품 연동 사용**: 체크 시 검색을 통해 기준이 될 마스터 상품을 선택합니다.
*   **재고 배수**: 마스터 수량 1개당 차감될 개수를 설정합니다. (예: 마스터가 1box(10개입)이고 현재 상품이 1개입 단품이면 배수를 0.1로 설정)

---

## 💰 4. 할인 및 구매 혜택

### 4.1 정기배송 할인
*   **정기배송 할인 사용**: 체크 시 해당 상품을 정기배송으로 구독할 때 적용할 추가 할인율(%)을 입력할 수 있습니다.

### 4.2 수량별 차등 할인 (Bulk Discount)
많이 살수록 싸게 파는 정책을 설정합니다.
*   **수량**: 할인 적용이 시작될 최소 구매 수량을 입력합니다.
*   **할인율**: 해당 수량 이상 구매 시 적용될 할인율(%)입니다.

---

## 🎁 5. 추가 증정 상품 (보너스 아이템)

특정 상품 구매 시 덤으로 제공할 아이템을 설정합니다.
1.  **상품 검색**: 증정할 상품을 찾아 추가합니다.
2.  **증정 방식**:
    *   **고정 수량(Fixed)**: 주문 수량과 관계없이 지정된 개수만 증정합니다.
    *   **비율 계산(Ratio)**: 구매 수량에 비례하여 증정품 개수가 자동으로 늘어납니다.

---

## ⚙️ 6. 주문 제한 설정
*   **최소/최대 주문 수량**: 한 번에 구매 가능한 범위를 제한합니다. (최대 0 입력 시 제한 없음)
*   **판매 단위**: 수량 선택 시 증감되는 단위를 설정합니다. (예: 5 입력 시 5, 10, 15... 단위로만 구매 가능)

---

## 📝 7. 상세 설명 및 완료
*   **상품 상세 설명**: 에디터를 사용하여 이미지, 표, 텍스트 등을 자유롭게 구성합니다.
*   **저장**: 하단의 `[등록 완료]` 또는 `[수정 완료]`를 클릭하여 저장합니다.
