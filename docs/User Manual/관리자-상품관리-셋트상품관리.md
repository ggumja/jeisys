# 🍱 관리자 - 상품관리 - 셋트상품관리 매뉴얼

이 문서는 제이시스 쇼핑몰의 **셋트 상품(번들 구성)**을 등록하고 관리하는 방법을 설명합니다. 일반 단품 등록 기능을 기반으로 하되, 셋트 상품 전용 프로세스로 독립되어 운영됩니다.

## 1. 셋트 상품이란?
여러 개의 상품을 하나로 묶어 판매하거나, 특정 수량 세트로 구성된 상품을 의미합니다. 일반 상품과 동일한 관리 항목(가격, 재고, 할인 등)을 가지지만 업무 효율을 위해 등록 과정이 분리되어 있습니다.

## 2. 등록 및 수정 프로세스

### 2.1 접근 방법
1. 관리자 메뉴의 **[일반상품관리]** 클릭
2. 상단의 **[셋트상품 등록]** (블루 컬러 버튼) 클릭

### 2.2 기본 정보 설정
*   **상품명**: 고객에게 노출될 세트 명칭을 입력합니다.
*   **카테고리**: 해당 세트가 속한 카테고리를 선택합니다.
*   **판매가/판매단위**: 세트 전체의 판매 가격과 구성 단위를 입력합니다.
*   **재고 관리**: 세트 전용 재고를 운영하거나 마스터 상품과 재고를 공유할 수 있습니다.

### 2.3 할인 및 옵션
*   **다량 구매 할인**: 세트 상품을 여러 개 구매할 때 적용되는 수량별 할인을 설정합니다.
*   **정기 배송**: 세트 상품의 정기 구독 할인 여부를 결정합니다.

## 3. 일반 상품과의 차이점
*   현재 등록 기능은 일반 상품과 동일하나, 향후 **세트 구성 내역 자동 표기**나 **번들 전용 재급유 로직** 등 셋트 상품 전용 기능이 이 페이지에 우선적으로 업데이트될 예정입니다.

---
> [!TIP]
