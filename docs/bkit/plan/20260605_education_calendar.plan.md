# bkit Plan - 교육 및 세미나 등록 기능 활성화 및 캘린더 구분 표시

- **작성일자**: 2026-06-05
- **담당자**: 봄(Bom)
- **목적**: 관리자 교육등록 기능 활성화, 교육/세미나 구분 필드 도입 및 캘린더 뱃지 컬러 차별화

## 1. 개선 배경 및 목표
- **현상**: 관리자 교육 일정 관리자 화면에서 "일정 등록" 버튼이 단순 마크업으로 작성되어 있으며 작동하지 않음.
- **목표**: 
  - 관리자 화면에서 일정 등록 및 편집 기능을 로컬 상태(`useState`) 기반으로 연동하고, 등록 모달(`Dialog`)을 제공.
  - 일정 등록 시 '교육(Education)'과 '세미나(Seminar)' 타입을 분기하여 입력 가능하게 처리.
  - 사용자 교육 캘린더 화면에서 교육 일정은 브랜드 컬러(`#21358D`), 세미나 일정은 보라색(`bg-purple-600`) 뱃지로 구분하여 가시성 개선.
  - 프로젝트의 UI/UX 가이드라인(순번 열 표시, 페이징 컴포넌트, 페이지당 행 수 선택) 준수.

## 2. 세부 구현 사양
- **데이터 구조 변경**:
  - `EducationSchedule` 인터페이스 내 `type: 'education' | 'seminar'` 속성 추가.
- **관리자 페이지 (`EducationManagementPage.tsx`)**:
  - `schedules` 로컬 상태 관리.
  - 등록 및 수정을 위한 `Dialog` 폼 구성 (일자, 구분, 장비, 일시, 장소, 정원, 강사, 상태 입력 가능).
  - UI 가이드라인 준수:
    - 리스트 테이블의 첫 번째 컬럼에 `No.` 컬럼을 적용하고 `(현재페이지 - 1) * 페이지당항목수 + (인덱스 + 1)` 공식 적용.
    - 하단에 페이징 컴포넌트 (`[이전] 1 2 3 [다음]`) 및 행 수 변경 셀렉트(10, 20, 50, 100) 구성.
- **사용자 페이지 (`EducationPage.tsx`)**:
  - 캘린더 날짜 박스에 나타나는 일정의 `type`에 따라 배경색 설정:
    - 교육: `bg-[#21358D] text-white`
    - 세미나: `bg-purple-600 text-white`
  - 하단 설명 영역에 세미나 뱃지 설명 추가.
  - 상세 팝업에서도 해당 일정이 세미나인지 교육인지 명확하게 표기.

## 3. 검증 계획
- 관리자 화면에서 일정이 등록, 수정, 삭제되며, 페이징에 따라 순번(No.)이 알맞게 재계산되는지 확인.
- 사용자 화면에서 달력의 뱃지 컬러가 교육/세미나 타입에 맞게 정확히 매핑되는지 검증.
