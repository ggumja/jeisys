# [Plan] 회원가입신청 반려한 회원 재승인 기능 추가

회원가입 신청 시 반려(REJECTED) 처리된 회원에 대해, 관리자가 다시 승인(APPROVED)할 수 있도록 재승인 기능을 추가합니다.

---

## 1. 개요 및 배경
- 현재 회원가입 신청을 반려(`REJECTED`)하면 회원의 상태는 `suspended`(정지)로 변경됩니다.
- 한번 반려된 회원은 관리자 화면에서 다시 승인할 수 있는 방법이 제공되지 않아, 수동으로 DB를 수정해야 하는 불편함이 있습니다.
- 따라서 반려 상태의 회원에 대해 '재승인' 기능을 구현하여 관리 효율성을 높입니다.

## 2. 변경 내용
### 2.1. 회원 상세 페이지 수정 (`src/pages/admin/MemberDetailPage.tsx`)
- `member.status === 'suspended'`인 경우 상단 관리 영역에 **[재승인]** 버튼을 노출합니다.
- [재승인] 버튼 클릭 시 `updateStatusMutation`을 통해 해당 회원의 상태를 `APPROVED`로 변경합니다.

### 2.2. 회원 목록 페이지 수정 (`src/pages/admin/MemberManagementPage.tsx`)
- 회원 목록에서 `suspended` 상태인 회원 옆에 **[재승인]** 버튼을 노출하여, 상세 페이지로 이동하지 않고도 목록에서 바로 재승인이 가능하도록 합니다.

## 3. 검증 계획
- 반려 상태인 회원의 상세 페이지에서 [재승인] 클릭 후 활성 상태로 변경되는지 확인
- 회원 목록 페이지에서 반려 상태인 회원 옆의 [재승인] 클릭 후 상태가 활성으로 바뀌는지 확인
