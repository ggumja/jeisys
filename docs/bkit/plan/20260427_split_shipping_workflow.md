# 분할 배송 및 배송 번들 워크플로우 구현 계획

## 개요
회원 분류 중 '분할 배송 허용' 옵션이 활성화된 고객을 대상으로, 주문 상품을 원하는 수량만큼 나누어 여러 배송지로 보낼 수 있는 '배송 번들(Bundle)' 기능을 구현함.

## 주요 기능
1.  **사용자/대리주문 UI**:
    *   회원 분류 권한 체크 후 '분할배송등록' 버튼 노출.
    *   상품별 잔여 수량 선택 및 배송지 정보를 입력하여 '계획된 번들(PLANNED Shipment)' 생성.
2.  **어드민 주문 관리**:
    *   주문 상세 페이지에서 배송 번들 단위로 상품 내역 그룹화.
    *   '대기 중인 번들'에 대해 로젠 API 연동 '부분발송하기' 기능 제공.
    *   송장 자동 발급 및 발송 상태(SHIPPED) 전환.
3.  **데이터 모델링**:
    *   `shipments` 테이블에 `status`('PLANNED', 'SHIPPED') 및 `shipping_info` 컬럼 추가.
    *   `shipment_items`를 통해 번들에 포함된 구성품 추적.
