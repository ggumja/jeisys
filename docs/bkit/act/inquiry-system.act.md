# PDCA Report: 1:1 문의 시스템 (1단계: 사용자 기능)

## 1. 완료 요약 (Summary)
프론트엔드 '커뮤니케이션' 메뉴의 **1:1 문의** 기능을 위한 핵심 인프라와 사용자 인터페이스 구현을 완료했습니다. Mock 데이터 기반의 기존 코드를 Supabase 실시간 연동 코드로 교체했습니다.

## 2. 주요 변경 사항 (Changes)
- **Data Service**: `inquiryService.ts`를 신규 생성하여 CRUD 인터페이스 표준화.
- **Inquiry Flow**:
    - **목록**: 실시간 DB 조회 및 상태(대기/완료) 표시.
    - **작성**: 문의 유형 선택, 제목/내용 입력, 비밀글 토글 기능 추가.
    - **상세**: 작성자 정보 노출, 답변 영역 렌더링, 비밀글 접근 제어(작성자/관리자 외 차단).
    - **관리자**: 관리자 전용 목록 페이지 연동 및 답변 등록/수정 모달 구현 완료.

## 3. 향후 과제 (Next Steps)
- **검증**: 비밀글 권한 정책(RLS)이 `is_secret` 필드를 정확히 반영하도록 SQL 업데이트.
- **알림**: 문의 답변 등록 시 사용자에게 알림(이메일/푸시) 발송 로직 검토.
